# Ocariq

> **Advancing Operations Control.**

Ocariq develops open, vendor-neutral architectures, specifications and reference implementations for **Operations Control**.

Ocariq establishes an open architectural foundation for coordinating Business Operations across heterogeneous cloud-native telecommunications environments.

## Principles

- Open by design
- Vendor neutral
- Architecture first
- Capability driven
- Community governed

## Ecosystem

The Ocariq ecosystem consists of four complementary projects:

- **Operations Control Architecture (OCA)** — Defines the architectural principles and operational model.
- **Operations Control Plane (OCP)** — Defines the reference runtime architecture.
- **Operations Control Runtime (OCR)** — Defines the software architecture and runtime components.
- **Operations Control Platform (OCPP)** — Provides the reference implementation.

Together these projects establish an architecture-first ecosystem for coordinating Business Operations across heterogeneous cloud-native telecommunications environments.
